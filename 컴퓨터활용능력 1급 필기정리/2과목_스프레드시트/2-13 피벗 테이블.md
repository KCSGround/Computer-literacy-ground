## 삽입 - 피벗 테이블
- 많은 양의 자료를 효율적으로 요약하여 나타냄
- 필터, 열, 행 값
- 값 필드 2개 이상 -> $\Sigma$ 추가

## 피벗 테이블 옵션과 추가기능
- [옵션]으로 '오류 값', '빈 셀', '행 & 열 총합계' 설정 가능
- 그룹으로 자료 그룹화
- 수식 추가로 값 필드 생성 O
- 특정 항목을 더블클릭하면 세부적인 데이터가 새로운 시트에 표시
- 원본 데이터 수정 시 새로고침

## 피벗 차트
- 피벗 테이블을 만들어야 생성 가능
- 피벗 차트 생성 후 피벗 테이블 삭제 시 일반 차트도 변경
- 분산형, 거퓸형, 주식형 X
- 피벗 차트 보고서에 필터를 적용하면 피벗 테이블 보고서에도 자동 적용
