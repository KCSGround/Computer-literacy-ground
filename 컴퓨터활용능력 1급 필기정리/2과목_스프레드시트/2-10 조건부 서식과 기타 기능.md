## 홈 - 조건부 서식
- 조건 만족 시 지정된 서식 적용
- 중복, 고유, 상 하위 값, 평균 기준
- 데이터 막대, 색조, 아이콘 집합 스타일
- 조건부 서식 충돌 시 우선순위 높은 것 적용

## 데이터 - 유호성 검사
- 입력된 데이터가 유효한지 검사
- 설정 : 제한 대상 지정
- 설명 메시지 : 자료 입력 시 설명이 보임
- 오류 메시지 : 유효하지 않은 데이터 입력 시 메시지 설정
- IME 모드 : 자료 입력 형태 설정
