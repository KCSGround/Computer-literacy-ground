## 데이터 제어어 (DCL)
- commit (완료, 승인), Rollback (취소), Grant (권한 부여), Revoke (권한 회수)

## 데이터 정의어 (DDL)
- DB를 생성하거나 자료 형태와 구조 등을 수정하며 데이터를 이용하는 방식을 정의하는 기능
- Create (생성) : create table 테이블명 (필드 형식 ..);
- Alter (구조 변경) : Alter table 테이블명
- Drop (삭제) : Drop table 테이블명;
