# 1. 에디터 종류

## Pico

워싱턴 대학의 Aboil Kasar가 개발한 유닉스 기반 텍스트 에디터

인터페이스가 윈도우의 <span style = "color:tomato">메모장</span>과 유사하여 간단

자유 소프트웨어 라이선스가 아니기 때문에 소스 수정 불가능

<span style = "color:tomato">GNU 프로젝트</span>에서는 pico의 복제 버전인 nano를 개발

---

## Emacs

<span style = "color:tomato">리처드 스톨만</span>이 개발

이후 <span style = "color:tomato">제임스 고슬링</span>이 LISP언어를 기반으로 업그레이드

---

## vi

<span style = "color:tomato">빌 조이</span>가 초기 BSD 릴리즈에 포함될 편집기로 만듬

명령모드, 입력모드, 편집모드로 구성

---

## vim

브람 무레나르가 개발

vi 편집기와 호환

다양한 색상을 이영하여 가시성 상승

---

<span style = "color:tomato">GTK+와 그놈 라이브러리</span>를 이용하여 개발
